# To-Do List App

A modern, responsive and production-style To-Do List web application built using HTML, CSS, and Vanilla JavaScript. Designed for real-life daily task management with a clean UI and smooth user experience.

🚀 Features
Add, edit, delete, and complete tasks
Task priority, due date, and category support
Search, filter, and sort tasks easily
Daily, weekly, and completed task views
Dark/Light mode
Local storage data persistence
Mobile-friendly responsive design

🛠️ Tech Stack
HTML5
CSS3
JavaScript

📂 Project Structure
/index.html
/style.css
/script.js

📌 Purpose
This project is built to improve productivity in daily life and demonstrate frontend development skills with real-world functionality.
